What I learned writing this page:

HTML5

According to W3, HTML stands for Hyper Text Markup Language, and it is the standard markup language for creating Web pages. HTML describes the structure of a Web page, consisting of a series of elements. These elements tell your browser how to display the content of the page.

Tags and attributes

Here, I used my first element tags in HTML. p stands for paragraph, and a stands for anchor. Paragraph organizes your text, images, videos, and other types of content. Anchor links other elements in the web. Two important attributes of Anchor are href (hyperlink reference), which points to a link, and target, which specifies whether the new page will open in the same tab, a different tab or a different window.

Here is a link to my LinkedIn profile. Because I attributed _blank to my target tag, it will open on a different tab. And here is a link to send me an email.

img was another tag that I learned. I used it to include images on this page, with alternative text for accessibility screen readers. Using the tags em for emphasis and strong for strong emphasis also help accessibility.

The last tags I learned were the ones that help me make lists in HTML. I made a list of the tags I learned in this short class. If I wanted my list to be ordered with numbers, I could use the ol (ordered list) tag instead.

  • html
  • head
  • meta
  • title
  • body
  • header
  • section
  • article
  • h1, h2, h3, h4, h5 and h6
  • img
  • p
  • a href
  • ul, ol and li

CSS3

CSS3 allows devs to create style rules for HTML elements, making it easier to change colors, sizes, fonts and more. All I had to do was link my css stylesheet to the index HTML page, using the tag link inside the head element.

Selectors and declarations

CSS3 tags are called selectors, because they select which HTML elements will be styled. Selectors can be a class or an id. In the stylesheet, I reference a class using a dot (.) and a class using a hashtag(#). Once I selected my HTML elements, I declare what I want them to look like. Declaration blocks come inside curly brackets, and each declaration inside the block is followed by a semicolon.

These were some attributes I used to declare the styles of my page:

  • background
  • color
  • font-size
  • font-style

As shown by font-size and font-style, I can make the attributes more specific by adding information with a dash (-).

I learned what a box model is, and that every box model has a margin (spaces between elements), a border (space around padding and content), a padding (space between the border and content) and some kind of content inside the padding. Margins can have the same value all around the border, or separate values for each one. Borders can be solid, dotted or dashed. The attribute border-radius was used to make the borders of each section round. The same attribute was used to make my photo at the top round, even though my original image file was rectangular (border-radius: 50%).

I also learned to style my lists with list-style-type, justify my text with text-align: justify, and keep my images inside their containers with max-width: 100%. Most importantly, I made my page fit any screen size by adding these declarations to my body element:

  • body {
  • max-width: 900px;
  • margin: auto;
  • }

Finally, I did some research to find out how to place my introduction text next to my profile photo, and I learned how to use display: inline-block to stop my photo from blocking an entire line, then width percentages to make that design responsive. Now you'll see a tiny profile pic if you're on your phone, or a bigger one if you're on a desktop monitor.
